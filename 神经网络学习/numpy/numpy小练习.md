   * ```python
     [[0, 1, 2],
      [3, 4, 5],
      [6, 7, 8],]
     ```

   * ```python
     [[0, 3, 6],
      [1, 4, 7],
      [2, 5, 8],]
     ```

6. 计算两个`3x3`的张量每个位置的最大值，并返回一个`3x3`张量。

   * 示例：

   * ```python
     # 张量1
     [[0, 1, 2],
      [3, 4, 5],
      [6, 7, 8],]
     # 张量2
     [[0, 3, 6],
      [1, 4, 7],
      [2, 5, 8],]
     # 结果
     [[0, 3, 6],
      [3, 4, 7],
      [6, 7, 8],]
     ```
